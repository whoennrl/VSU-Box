    Разработка расширений для VSU Box

    Расширения (аддоны) позволяют добавлять собственный функционал прямо в интерфейс VSU Box: создавать новые экраны, добавлять кнопки в меню, отображать данные расписания, сохранять пользовательские данные и многое другое.

    Как это работает

    Код расширения выполняется в изолированной sandbox="allow-scripts" iframe-песочнице. Прямого доступа к DOM приложения нет — всё взаимодействие происходит через VSUBoxSDK по протоколу postMessage.

    Жизненный цикл расширения

    1. Разработчик создаёт расширение в Меню → Режим разработчика и загружает файлы
    2. Указывает нужные разрешения (schedule, user, storage и др.) в настройках аддона
    3. Отправляет на проверку администратору
    4. После одобрения пользователи видят аддон в Магазине
    5. При установке пользователь подтверждает запрошенные разрешения нажатием «Разрешить и установить»
    6. Приложение автоматически запускает аддон при каждом старте через SDK

    Что может расширение

    • Создавать полноэкранные интерфейсы и добавлять кнопки в меню навигации
    • Использовать готовые UI-компоненты: кнопки, карточки, списки, изображения, поля ввода
    • Принимать события от UI-элементов (клики, ввод текста) через систему actions
    • Показывать модальные диалоги: тост, alert, confirm, ввод текста, выбор оценки
    • Читать расписание пользователя (пары, время, аудитории, преподаватели)
    • Получать имя и роли текущего пользователя
    • Хранить произвольные данные на сервере через key-value хранилище
    • Читать и записывать файлы в персональную директорию пользователя (до 20 МБ)
    • Подписываться на системные события: установка PWA, видимость приложения
    • Делать HTTPS-запросы на сторонние серверы через sdk.http.fetch() — addon-токен подставляется автоматически
    • Обмениваться событиями с другими расширениями через sdk.events.emit()
    • Запускать логику при каждом старте приложения через разрешение autostart
    • Отправлять push-уведомления пользователю с вашего бэкенда через API и addon-токен

    Важно

    Прямые сетевые запросы из sandbox заблокированы (fetch, XMLHttpRequest). Для HTTP-запросов на внешние серверы используйте sdk.http.fetch() — он безопасно проксирует запрос с подстановкой токена.

    Быстрый старт

    VSUBoxSDK.ready(async (sdk) => {
    const screenId = await sdk.ui.createScreen({ title: "Мой аддон" })

    await sdk.ui.setContent(screenId, `
        <div style="padding:24px;font-family:Nunito,sans-serif">
            <h2>Привет из расширения!</h2>
        </div>
    `)

    await sdk.ui.addMenuButton({ label: "Мой аддон", screenId, icon: "star.fill" })
})
