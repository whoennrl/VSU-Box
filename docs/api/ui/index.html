    sdk.ui — Интерфейс

    Управление экранами, навигацией, уведомлениями и компонентами. Разрешений не требует.

    Экраны

    sdk.ui.createScreen({ title })

    Создаёт новый полноэкранный интерфейс с навбаром и кнопкой «Назад».

    Возвращает: Promise<screenId: string>

    const screenId = await sdk.ui.createScreen({ title: "Заметки" })

    sdk.ui.setContent(screenId, html)

    Устанавливает HTML-содержимое тела экрана в изолированной iframe-песочнице. Поддерживается любой валидный HTML, включая теги <script>, <style> и изображения — всё выполняется внутри изолированной среды без доступа к DOM приложения.

    await sdk.ui.setContent(screenId, `
    <div style="padding:20px;font-family:Nunito,sans-serif">
        <h2 style="margin-top:0">Мои заметки</h2>
        <p style="color:#888">Пока пусто</p>
    </div>
`)

    sdk.ui.navigate(screenId)

    Показывает указанный экран (скрывает все остальные).

    Вызов при старте

    Вызов navigate() внутри ready() при старте приложения игнорируется — экран откроется только когда пользователь нажмёт кнопку в меню.

    sdk.ui.setTitle(screenId, title)

    Меняет заголовок уже созданного экрана (текст в навбаре).

    sdk.ui.updateElement(screenId, elementId, html)

    Обновляет содержимое одного элемента внутри content-iframe по его id — без полной перерисовки экрана. Удобно для частых обновлений (таймер, счётчик, статус).

    Параметр Тип Описание
    screenId string Идентификатор экрана
    elementId string id элемента внутри HTML-контента
    html string Новый HTML для innerHTML элемента

    Возвращает: Promise<boolean> — false если экран или элемент не найден.

    await sdk.ui.setContent(sid, `
    <div style="padding:20px;font-family:Nunito,sans-serif">
        <div id="counter" style="font-size:48px;font-weight:800;text-align:center">0</div>
    </div>
`)

let n = 0
setInterval(async () => {
    n++
    await sdk.ui.updateElement(sid, "counter", String(n))
}, 1000)

    sdk.ui.back()

    Возвращает пользователя на главный экран приложения. Кнопка «<» в навбаре делает то же самое автоматически.

    sdk.ui.getTheme()

    Возвращает: Promise<"light" | "dark">

    Возвращает текущую тему оформления. Учитывает как настройки пользователя, так и системные предпочтения (при выборе «Авто»).

    // Полный цикл: создать → заполнить → открыть
const sid = await sdk.ui.createScreen({ title: "Заметки" })

await sdk.ui.setContent(sid, `
    <div style="padding:20px;font-family:Nunito,sans-serif">
        <h2 style="margin-top:0">Мои заметки</h2>
    </div>
`)

await sdk.ui.navigate(sid)

    Меню навигации

    sdk.ui.addMenuButton({ label, screenId, icon? })

    Добавляет кнопку в раздел «Меню» приложения (боковой сайдбар на десктопе, нижнее меню на мобиле). При нажатии открывает указанный экран.

    Параметр Тип Описание
    label string Текст кнопки
    screenId string Идентификатор из createScreen
    icon string Имя иконки SF Symbol, например "star.fill". Если не указано — используется "exclamationmark.circle.fill" 
    const sid = await sdk.ui.createScreen({ title: "Аналитика" })
await sdk.ui.addMenuButton({
    label:    "Аналитика",
    screenId: sid,
    icon:     "chart.bar.fill"
})

    Уведомления и диалоги

    sdk.ui.toast(message, duration?)

    Показывает всплывающее уведомление снизу экрана. Автоматически скрывается через duration миллисекунд (по умолчанию 2500).

    await sdk.ui.toast("Данные сохранены!", 3000)

    sdk.ui.showAlert(message)

    Показывает системный диалог с сообщением и кнопкой ОК. Выполнение ждёт закрытия диалога.

    sdk.ui.showConfirm(message)

    Показывает диалог с кнопками ОК/Отмена.

    Возвращает: Promise<boolean> — true если пользователь нажал ОК.

    const ok = await sdk.ui.showConfirm("Сбросить все данные?")
if (ok) {
    await sdk.storage.delete("data")
    sdk.ui.toast("Данные удалены")
}

    Обратная связь

    sdk.ui.showInput(opts)

    Показывает красивый модальный диалог с полем ввода текста.

    Возвращает: Promise<string | null> — введённое значение, или null если пользователь нажал «Отмена».

    Параметр Тип Описание
    title string Заголовок диалога
    message string Подсказка под заголовком
    placeholder string Плейсхолдер поля
    default string Значение по умолчанию
    type string Тип input (text, number, email и т.д.)
    confirm string Текст кнопки подтверждения (по умолчанию «ОК»)
    cancel string Текст кнопки отмены 
    const name = await sdk.ui.showInput({
    title:       "Как вас зовут?",
    placeholder: "Введите имя...",
    confirm:     "Сохранить",
})
if (name) await sdk.storage.set("name", name)

    sdk.ui.share(opts)

    Открывает нативный диалог шаринга (Web Share API). Если браузер не поддерживает нативный шаринг — показывает кастомный bottom sheet с доступными действиями.

    Возвращает: Promise<{ shared: boolean, action?: string, reason?: string }>

    Параметр Тип Описание
    title string Заголовок для нативного диалога и превью
    text string Основной текст
    url string URL для «Поделиться» / «Скопировать ссылку»
    image string URL или data URI изображения (PNG/JPEG). Предлагает «Сохранить изображение»
    filename string Имя файла при сохранении изображения (по умолчанию "image.png") 
    // Поделиться ссылкой
const result = await sdk.ui.share({
    title: "Расписание на неделю",
    text:  "Моё расписание в VSU Box",
    url:   "https://vsu-box.ru"
})
if (result.shared) sdk.ui.toast("Готово!")

// Поделиться изображением (data URI или внешний URL)
await sdk.ui.share({
    title:    "Скриншот расписания",
    image:    "data:image/png;base64,...",
    filename: "schedule.png"
})
    result.action Описание
    "share" Пользователь поделился через нативный диалог
    "save" Пользователь сохранил изображение
    "copy_url" Ссылка скопирована в буфер обмена
    "copy_text" Текст скопирован в буфер обмена

    sdk.ui.loadHtml(path)

    Загружает HTML-файл из хранилища расширения на сервере (файл должен быть загружен через DevMode вместе с entry_file).

    Возвращает: Promise<string> — содержимое файла.

    Параметр Тип Описание
    path string Путь к файлу относительно корня расширения, например "screens/home.html" 
    const html = await sdk.ui.loadHtml("screens/main.html")
await sdk.ui.setContent(screenId, html)

    Когда это нужно

    Используйте loadHtml() если HTML-шаблоны стали слишком большими для инлайна в JS-коде — разнесите их по файлам и загружайте по мере необходимости.

    sdk.ui.showRating(opts)

    Показывает диалог выбора звёздной оценки.

    Возвращает: Promise<number | null> — число от 1 до stars, или null при отмене.

    Параметр Тип Описание
    title string Заголовок
    stars number Количество звёзд (2–10, по умолчанию 5)
    confirm string Текст кнопки
    cancel string Текст кнопки отмены 
    const rating = await sdk.ui.showRating({ title: "Оцените расширение", stars: 5 })
if (rating !== null) {
    sdk.ui.toast(`Спасибо за оценку: ${rating} из 5!`)
}

    События UI-элементов

    Реагируйте на клики и ввод прямо в HTML-содержимом экрана. Добавьте атрибут data-sdk-action="имя" на любой элемент — SDK автоматически передаст событие в ваш код.

    sdk.ui.onAction(actionId, callback)

    Регистрирует обработчик клика на элемент с data-sdk-action="actionId". Вызов sdk.ui.navigate() автоматически активирует прослушивание actions на экране.

    Параметр Описание
    actionId Строка-идентификатор
    callback(value) value = значение <input> внутри элемента, или data-sdk-value

    Как передаётся value

    • Для кнопки: value — значение первого <input> на экране
    • Для <input data-sdk-action>: value — текущее значение самого поля при клике
    const sid = await sdk.ui.createScreen({ title: "Форма" })

await sdk.ui.setContent(sid, `
    <div style="padding:20px;font-family:Nunito,sans-serif">
        <input data-sdk-action="name-input" type="text"
               placeholder="Введите имя..."
               style="width:100%;border:none;background:rgba(0,0,0,0.06);
                      border-radius:12px;padding:12px 14px;font-size:14px">
        <button data-sdk-action="save-btn"
                style="margin-top:12px;background:rgb(0,122,255);color:white;
                       border:none;border-radius:12px;padding:12px 24px;
                       font-size:14px;font-weight:700;cursor:pointer;width:100%">
            Сохранить
        </button>
    </div>
`)

sdk.ui.onAction("save-btn", async (value) => {
    sdk.ui.toast(`Сохранено: ${value}`)
})

// navigate() автоматически активирует прослушивание actions
await sdk.ui.navigate(sid)

    Иконки SF Symbols

    Метод sdk.ui.icon() создаёт иконку из набора SF Symbols в стиле приложения. Иконки отрисовываются через CSS mask-image — цвет задаётся произвольно.

    sdk.ui.icon(name, opts?)

    Возвращает: string (HTML)

    Параметр Тип Описание
    name string Имя иконки без расширения, например "star.fill"
    opts.size number Размер в px (по умолчанию 22)
    opts.color string Цвет CSS (по умолчанию "currentColor") 
    // Иконка со звездой синего цвета
const starIcon = sdk.ui.icon("star.fill", { size: 24, color: "rgb(0,122,255)" })

// Иконка в карточке
const row = `
    <div style="display:flex;align-items:center;gap:12px;padding:14px;
                background:white;border-radius:14px;margin:6px 0">
        ${sdk.ui.icon("calendar.fill", { size: 22, color: "rgb(0,122,255)" })}
        <span style="font-size:15px;font-weight:600">Расписание</span>
    </div>
`

    Готовые UI-компоненты

    Методы sdk.ui.components.* возвращают строки HTML с уже готовыми стилями в стиле VSU Box. Вставляйте их через setContent() или конкатенацию.

    Доступные компоненты

    Метод Параметры Описание
    components.button(text, opts) id, primary: bool Кнопка. id становится data-sdk-action
    components.card(opts) title, desc, icon (URL), actionId Карточка с иконкой, заголовком и подписью. actionId делает кликабельной
    components.text(content, opts) size, bold, muted, center Параграф текста с гибкими стилями
    components.heading(text, level) level: 1 | 2 | 3 Заголовок с размерами 22 / 18 / 15 px
    components.image(src, opts) alt, radius, width Изображение с закруглёнными углами
    components.input(opts) id, placeholder, type, value Стилизованное поле ввода. id → data-sdk-action
    components.list(items) Массив: { title, subtitle, actionId } или строки Сгруппированный список в карточке (как iOS UITableView)
    components.badge(text, opts) color, textColor Цветной бейдж / пилюля
    components.divider() — Горизонтальный разделитель
    components.spacer(height) height: number (px) Вертикальный отступ

    Пример использования компонентов

    const c = sdk.ui.components  // псевдоним для краткости

const html =
    c.heading("Мой список") +
    c.list([
        { title: "Настройки",   subtitle: "Тема, язык",    actionId: "open-settings" },
        { title: "О программе", subtitle: "Версия 1.0.0",  actionId: "open-about" },
    ]) +
    c.spacer(12) +
    c.button("Сохранить", { id: "save", primary: true })

await sdk.ui.setContent(screenId,
    `<div style="padding:20px;font-family:Nunito,sans-serif">${html}</div>`
)

sdk.ui.onAction("open-settings", () => sdk.ui.navigate(settingsScreenId))
sdk.ui.onAction("save", () => sdk.ui.toast("Сохранено!"))
